<h1>Filter Tutorial</h1>
<p>Upload an audio file and apply different filters to the track.</p>

<p><b>Lowpass Filter</b><br>
<p>Standard second-order resonant lowpass filter with 12dB/octave rolloff.
Frequencies below the cutoff pass through; frequencies above it are attenuated.
Cutoff frequency is set at 80 Hz for this lesson</p><br>
<p><b>Highpass Filter</b><br>
<p>Standard second-order resonant highpass filter with 12dB/octave rolloff.
Frequencies below the cutoff are attenuated; frequencies above it pass through.
Cutoff frequency is set at 1000 Hz for this lesson</p><br>
<p><b>Bandpass Filter</b><br>
<p>Standard second-order bandpass filter. Frequencies outside the given range
of frequencies are attenuated; the frequencies inside it pass through.
The center of the frequency range is set at 500 Hz for this lesson. The Q
value determines the width of the band (lower Q-value means wider). The Q-value
is set at 2 for this lesson.</p><br>
<p><b>Lowshelf Filter</b><br>
<p>Standard second-order lowshelf filer. Frequencies lower than the frequency
get a boost, or an attenuation; frequencies over it are unchanged.
Shelf Frequency is set at 80 Hz for this lesson, below which the frequencies
are boosted by 10 dB</p><br>
<p><b>Highshelf Filter</b><br>
<p>Standard second-order highshelf filer. Frequencies higher than the frequency
get a boost or an attenuation; frequencies lower than it are unchanged.
Cutoff frequency is set at 1000 Hz for this lesson, above which the frequencies
are boosted by 10 dB</p><br>
<p><b>Allpass Filter</b><br>
<p>Standard second-order allpass filter. It lets all frequencies through, but
changes the phase-relationship between the various frequencies. The frequency
where the center of the phase transition occurs is set at 500 Hz. The Q-value,
which controls how sharp the transition is at the medium frequency, is set to
2 for this lesson (The larger this parameter is, the sharper and larger the
transition will be.)</p>
